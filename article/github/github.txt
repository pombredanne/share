= GitHub =

想用 link:../git/git.html[Git] 不一定得要自己架設主機才行，網路上有許多免費的代管服務，這裡所要介紹的就是赫赫有名的 GitHub /'ɡɪtˌhʌb/。許多知名的專案（例如 Twitter、Facebook、Digg、jQuery 等）都代管在 GitHub 上面，它除了免費提供 public repositories 之外（private repositories 則需要付費），還提供有 issue tracking、wiki、code review 等協同開發所需要的服務，再加上許多設計都是社交導向的（是否注意到 GitHub 的標語是 SOCIAL CODING？），使得 GitHub 很適合用來整合分散在各地的開發者，共同完成一個專案。

== 雜記 ==

 * personal/organization account，後者跟 GitHub Enterprise 把 GitHub 執行在企業內的做法沒有關係。
 * fork, pull request
 * issue tracking, wiki, code review
 * public/private repositories，其中 public repository 是免費的
 * GitHub 的 markup 以 Markdown 最為優先，但也https://github.com/github/markup#readme[支援 AsciiDoc]。
 * repository (repo) 看起來像是 remote git directory (local repo)。通常一個 repo 對應一個 project，這樣 project pages 才不會打架。
 * Account Settings 裡可以設定 Jobs Profile，整合求職的功能很特別。
