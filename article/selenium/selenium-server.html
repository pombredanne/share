Selenium / (RC) Server

Selenium Server 其實是泛稱之前的 Selenium 1.x 的 Selenium Remote Control (RC)，跟 Selenium 2.x 的 Selenium WebDriver 不同。

在 Selenium RC/WebDriver 合併之後，Selenium Server 2.x 做為 1.x 的 drop-in replacement，多了 WebDriver 的支援，但向下還是相容於 Selenium RC。會這麼做有很大的比例上是因為，目前 (2012-07) Selenium RC 支援的程式語言跟瀏覽器的種類，還是比 Selenium WebDriver 來得多。

Selenium 1 is still actively supported (mostly in maintenance mode) and provides some features that may not be available in Selenium 2 for a while, including support for several languages (Java, Javascript, Ruby, PHP, Python, Perl and C#) and support for almost every browser out there.
