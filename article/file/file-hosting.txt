= link:file.html[Files] / 檔案代管（Hosting）服務 =

目前可行的方案有：

 * http://www.dropbox.com/[Dropbox]
 * https://one.ubuntu.com/[Ubuntu One]

雖然 Ubuntu One 目前也朝跨平台的客戶端努力中，但 Dropbox 相對比較老牌，客戶端平台的支援比較廣泛。
