:title: 安裝 Firefox

== Windows ==

從http://www.mozilla.org/en-US/firefox/new/[官網]下載（`Firefox Setup <version>.exe`）後安裝即可。

== Mac OS X ==

 * 到http://www.mozilla.org/en-US/firefox/new/[官網]按 Download，就會開始下載 `Firefox 11.0.dmg`。
 * 將 Firefox 拖到 `/Applications` 即可。
