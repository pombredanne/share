Android / Automation / 工具（Tools）

目前抬面上比較知名的工具有：

  • monkeyrunner

    • 測試用 Python 寫

  • UI Automator

  • Calabash Android - 內部包裝 Robotium

  • Robotium

    • 用 instrumentation 技術，而且要寫成 JUnit，跟 Robot Framework 難以整合。

    • 不過 Robotium 的 Remote Control 可以期待，跟 NativeDriver 在做的事很像

  • Robotium RC

  • MonkeyTalk - 氣勢還不錯，但還是比 Calabash 弱了一點？

  • Sikuli

    • 輔以 monkeyrunner 可以做一些額外的控制，例如按下硬體鈕。

    • 搭配 emulator 不是問題，但如果是 Android device 的話要被 root 才行。

  • NativeDriver - 已經停止開發了

    • 一樣是走 Instrumentation，但 test code 是跑在電腦上，有機會跟 Robot Framework 整合。

    • 可以補足 monkeyrunner 不能取得文字的缺憾。

其他還有：

  • aster - Android System Testing Environment and Runtime - 0xlab 開發的專案

  • (DataDriven for Robotium and Selenium (DFRS) - 把 Robotium 包裝成 keywords，但有自己的 test runner。

  • Robolectric: Unit Test your Android Application

  • SeeTest of Experitest - 手機接在自己的電腦上，錄製後可再輸出成 QTP, Python 等 script，專人聯絡才能取得 10 天試用。
