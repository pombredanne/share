Python / Types

基本上，built-in type 都是 new-style class，可以從 types 取得對應的 type object：（括號內的是別名）

  • None (NoneType)

  • int (IntType)

  • long (LongType)

  • float (FloatType)

  • str (StringType)

  • unicode (UnicodeType)

  • tuple (TupleType)

  • list (ListType)

  • dict (DictType)

  • FunctionType

  • ModuleType 

>>> import types
>>> type(int), type(file), type(None)
(<type 'type'>, <type 'type'>, <type 'NoneType'>)
>>> int == types.IntType, file == types.FileType, type(None) == types.NoneType
(True, True, True)

另外還有一個很特別的 StringTypes，方便用來做字串的 type checking。
