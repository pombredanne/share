Screen / Scrollback/Copy Mode

C-a [ - 進入 scrollback mode，按 Esc 可以離開。在這個模式下，除了一般的方向鍵、PgUp、PgDn、Home、End 可以用來翻頁之外，還有：

  • /<keyword>

  • ? <keyword>

  • <space> - 選取複製的範圍。

  • C-a ] - 貼上（並離開 scrollback mode）。

  • h/j/k/l - 跟 Vi 一樣

  • G - 移動到 buffer 結尾處。

  • g - 移動到 buffer 開始處。

由於進入 scrollback mode 後可以 copy/past，所以也稱做 copy mode。
