Screen / 在現有的 Session 裡開新 Window 執行 Command

screen -S <session_name> -X screen [-t <title>] <external_command> [<args>]

在名叫 session_name 的 screen session 裡開一個新 window 執行 external_command，並將 window title 改為 title。
