Books / LibriVox

LibriVox 唸做 [ˈlibrɪˌvɑks]，一個在網際網路上的有聲書圖書館，目標是要將所有公有領域（public domain）的書籍都錄製成有聲書，放在 Internet 上供人免費取用。Creative Commons 的副總裁 Mike Linksvayer 曾說："perhaps the most interesting collaborative culture project this side of Wikipedia"。

按維基百科的說法：創立版權制度的初衷是：藉由給予創作者一段時期的專有權利，作為（經濟）刺激，以鼓勵作者從事創作。當專有權利期間屆止，作品便進入公有領域。公有領域的作品由於沒有專屬權利人，因此公眾有權自由使用它們。

書本由全球各地的志工錄製並上傳錄音檔（.mp3 或 .ogg），由 LibriVox community 確認（proof-listened）過後，就可以在 LibriVox 上公開取用（錄音檔由 Internet Archive 代管）。目前（2011-08）約有 4500 多本書完成錄音，包含 30 幾國語言的書籍（以英文居多）。

透過目錄找到書本之後，可以分別下載多個錄音檔，也可以點選 Zip file of the entire book 的連結，一次下載所有的錄音檔。畫面上通常都提供有：

  • Gutenberg e-text 的連結，可以進一步取得書籍的文字內容（HTML、EPUB 等不同格式）。

  • CD booklet 的連結，用以將錄音檔燒錄並包裝成精美的 CD。

Project Gutenberg（古騰堡工程）也跟公有領域的書籍有關，是世界上最早的數位圖書館，它致力將公有領域的書籍數位化，使文化能夠保存下來。
