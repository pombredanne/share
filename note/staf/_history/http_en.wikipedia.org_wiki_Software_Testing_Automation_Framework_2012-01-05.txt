[edit] Introduction

   The Software Testing Automation Framework (STAF) is an open source
   project that enables users to create cross-platform, distributed
   software test environments.

[edit] Services

   STAF includes a number of services that provide specific functionality.
   The most prominent of these is called the STAf eXecution engine (STAX),
   which executes test scripts. Other services provide cron, file system,
   inter-process communication, e-mail, and HTML support, among others.
   The documentation includes instructions and guidelines for developers
   to generate their own custom services, as well.

[edit] Support

   IBM supports users of STAF through extensive online documentation and
   user forums, accessible via the STAF website.
