Vim / 安裝

Windows

從官網下載 gvim<version>.exe 安裝即可。過程中可以勾選 Create .bat files for command line use

預設會安裝到 C:\Program Files (x86)\Vim

Ubuntu

Ubuntu 預設只裝有 vim-common 跟 vim-tiny，要額外安裝 vim 套件之後，操作起來會比較方便。（例如可以用方向鍵移動游標）
