Vim / Snippets

Snippets 有很多種做法，早期 snippetsEmu 最常被提及，不過後來似乎整個倒向 snipMate。

  1. 下載 snipMate.zip，然後將它解壓縮到 ~/.vim 即可。 

    $ unzip snipMate.zip -d ~/.vim

  2. 做幾個簡單的測試 

    $ vi index.php
for<tab>

for ($i = 0; $i < count; $i++) {
         // code...
}
