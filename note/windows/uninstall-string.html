取得某個應用程式的 Uninstall String

參考資料

  • Obtain an uninstall string for any application - Spiceworks - HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall 從 display name 找出 UninstallString 或 QuietUninstallString

  • Uninstall programs that do not appear in the add or remove (2009-03-28) - 取得 uninstall string 後，再用 MsiExec.exe /I 移除。
