= Windows 內建的環境變數 =

常用的環境變數有：

 * `%APPDATA%` - 應用程式存放各使用者專屬資料的地方，預設是 `C:\Documents and Settings\{username}\Application Data`（Windows XP） 或 `C:\Users\{username}\AppData\Roaming`（Windows Vista 之後）。
 * `%USERPROFILE%` - 使用者家目錄，預設是 `C:\Documents and Settings\{username}`（Windows XP） 或 `C:\Users\{username}`（Windows Vista 之後）。
 * `%ProgramFiles%` - 應用程式安裝的位置，預設是 `C:\Program Files`。
 * `%ProgramFiles(x86)%` - 這個環境變數只會出現在 64-bit，做為安裝 32-bit 應用程式的位置，預設是 `C:\Program Files (x86)`。
 * `%ProgramData%` - 應用程式存放資料的地方，預設是 `C:\ProgramData`。這是 Windows Vista 之後才有的資料夾，因為 `%ProgramFiles%` 的權限控管變得更加嚴謹，為了防止程式被惡意竄改，就算是應用程式本身也不能對安裝目錄做寫入，也因此才會將應用程式的資料拆開來存放。
 * `%TEMP%` 和 `%TMP%` - 暫存檔案存放的地方，預設是 `C:\Documents and Settings\{username}\Local Settings\Temp`（Windows XP） 或 `C:\Users\{Username}\AppData\Local\Temp`（Windows Vista 之後）。
