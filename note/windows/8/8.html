學習筆記

  • Windows 8 的版本分為 ??? 外加 Windows 8 on ARM (WOA)，也會跑在手機上??

    • 支援 ARM CPU，是為了改善 Windows 在平板電腦上的表現。但 WOA 並不能直接跑 x86/64 的應用程式。

    • 除了 Windows 8 之外，另外還有 Windows Server 8 (WS8)，後來更名為 Windows Server 2012

  • 除了傳統介面之外，也支援新的 Metro UI，使用者可以隨時做切換。

    • Tile-based Start screen；從 Windows 95 開始有的 Start Menu 完全在 Windows 8 上消失了，另外功能列圖示 Metro 化。

    • 源自 Windows Phone 7，專為觸控式螢幕設計，

    • 每一個 tile 都代表一個應用程式，上面可以顯示動態的資訊。

    • Metro-style （相對於 desktop 的說法） applications 都是全螢幕，可以從 Windows Store 上直接購買，但會鎖定特定的地區（有些程式只有特定地區的使用者才能用??）

  • Windows 8 將內置防毒程式??

  • Windows 7 的軟體都可以跑在 Windows 8 上。

  • 內建 Hyper-V 虛擬化軟體??

  • 內建 IE 10，同時提供有 desktop/Metro-style 兩個版本。其中 Metro-style 不支援 plugin 跟 ActiveX，但 desktop 版本則繼續支援。

  • 無論在 Desktop 或 Start Screen，把滑鼠移到左下角可以在 Desktop/Start Screen 間切換（直接按 Windows 國旗鍵更快），移到右下角可以叫出 Search/Share/Start/Devices/Settings 的選單。（重新）關機可以從 Settings > Power > Shut down / Restart

  • 在 Start Screen 直接輸入，會直接叫出 Search App 的功能。例如輸入 notepad 就可以叫出以前慣用的 Notepad

  • Win+Q - 直接搜尋 Apps。

  • 移除應用程式可以在 Tile 上按右鍵，選 Uninstall 即可。 *
