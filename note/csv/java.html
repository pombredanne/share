CSV / Java

目前 (2009-08-27) 普遍推薦使用 Super CSV, 不過後起的 Commons CSV 納編了幾個現有的 libraries (其中最重要的是 Open CSV), 有要統一 CSV libraries 之勢…

  • Open CSV - 已被納編到 Commons CSV, 剛於 2009-08-24 釋出 2.0 版; 有發行到 Maven central

  • Super CSV - 雖然呼聲很高, 但沒有發行到 Maven repository

  • Commons CSV - 目前 2009-08-27 還在發展中, 它試圖要統一 CSV 存取的介面

  • csvbind

  • CSVObjects

  • JSefa - A simple library for stream-based serialization of java objects to XML, CSV, and FLR (extensible to other formats) and back again using an iterator-style interface independent of the serialization format.

  • csvio - 功能非常的陽春

  • CSVBeans - 2007-11 就沒再開發了

  • CsvJdbc - A JDBC driver for CSV files. 2001-01 後就沒再更新過了

  • xlSQL - 它也可以用來讀寫 CSV 格式, 不過已經停止開發了

  • FileHelpers - A .NET Equivalent to Java Super CSV
