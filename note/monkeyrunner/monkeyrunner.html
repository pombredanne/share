monkeyrunner

monkeyrunner 跟會送出亂數 events 的 Monkey 不同。monkeyrunner 是透過 API 送出特定的 command/event。

  • monkeyrunner 提供 Python API 來操控 device/emulator，主要用來做 functional test。

  • monkeyrunner 的底層是用 Hierarchy Viewer 跟 ChimpChat。
