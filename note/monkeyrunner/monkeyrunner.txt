= monkeyrunner =

NOTE: monkeyrunner 跟會送出亂數 events 的 http://developer.android.com/guide/developing/tools/monkey.html[Monkey] 不同。monkeyrunner 是透過 API 送出特定的 command/event。

 * monkeyrunner 提供 Python API 來操控 device/emulator，主要用來做 functional test。
 * monkeyrunner 的底層是用 link:monkeyrunner-hierarchyviewer.html[Hierarchy Viewer] 跟 link:monkeyrunner-chimpchat.html[ChimpChat]。

 * 不同於 monkey 透過 `adb shell` 送亂數的 command/event，但 monkeyrunner 是透過 API 送特定的 command/event。聽說這個 API 是 Android 2.1 之後就支援的??
 * Regression testing - monkeyrunner can test application stability by running an application and comparing its output screenshots to a set of screenshots that are known to be correct. 但用整個 screenshot 做比對並不務實，因為畫面上可能會有時間、廣告等會變動的東西
 * 可以同時控制多台 emulator/device，叫起 emulator 時，還能套用不同的 configuration。
 * 安裝並執行 application 或 test package、送出 keystrokes 或 touch events、截圖等。
 * 執行期用 Jython，可以同時叫用 Java/Python library，輔以呼叫外部程式 `adb`，可以做的事情就更多了。
 * 用 `MonkeyRunner` 連接設備，再透過回傳的 `MonkeyDevice` 操作，過程中完全用坐標來定位，這是它最大的弱點。
