monkeyrunner: Touch Gestures（觸控手勢）

monkeyrunner API 裡跟觸控相關的操作只有兩個，都集中在 MonkeyDevice：

  • touch(x, y, type) - 在 (x, y) 觸發單一個 touch event，用法可以參考輕觸（Tap）跟長按（Long Press）。

  • drag(start, end, duration=1.0, steps=10) - 模擬手指頭從 start 滑動到 end 的動作（touch-hold-move），用法可以參考滑動（Pan）。

上述的 drag gesture 跟 Drag-and-Drop (DnD) 沒有直接的關聯，雖然它可以用來完成 DnD。基本上，touch() 用在手指頭碰觸到螢幕後沒有改變位置的狀況，但如果有改變位置（滑動）的話就得用 drag()。

雖然相關的 API 並不多，但只要能 決定畫面上的座標值，就可以完成許多常見的觸控手勢：

  • 輕觸（Tap）跟長按（Long Press）

  • 掃動（Flick）

  • 拖放（DnD）

  • 滑動（Pan）

  • 捲動（Scroll）

不過目前 API 似乎還不支援多點觸控：

  • 縮放（Pinch/Spread）

  • 旋轉（Rotate）
