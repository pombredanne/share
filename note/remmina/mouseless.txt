:title: [Remmina] Mouseless

下面 Host 鍵預設是右邊的 Ctrl，可以從 Preferences > Keyboard 修改。

 * Host - 把所有鍵盤的按鍵都送到遠端，這樣像 Alt + F4 或 Ctrl + Alt + Del 等組合鍵就會直接作用在遠端那台電腦。
+
TIP:全螢幕模式下，可以先按一下 Host，接下來的組合鍵就會送給本地端的系統。要來回在本機其他桌面間切換時尤其方便。
+
 * Host + Left / Right - 左右切換不同的 tab （但只能在同一個 group 的不同連線間切換）。
+
如果要切換不同視窗的話，用link:../ubuntu/mouseless.html[系統內鍵]的 Alt + ` 即可。
+
 * Ctrl + O - 開啟連線。
 * Host + F4 - 關閉連線。
 * Host + F9 - 將連線畫面縮小，顯示所有電腦的清單。得按 Alt + ` 在不同視窗間切換才能叫回來？
 * Host + F - 切換全螢幕。
