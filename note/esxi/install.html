ESXi / 安裝

目前（2012-10）最新版是 VMware vSphere Hypervisor 5.1，下載頁面提供了 license keys 跟：

  • VMware ESXi Installable (ISO；約 301M) - 除了安裝之外，也可以用來昇級舊版。

  • VMware Tools

  • VMware vSphere Client

    1. 可以先看過 VMware Compatibility Guide 檢查硬體配備是否合乎要求。

    2. 利用 UNetbootin 製作 ESXi 的 USB 開機碟。

    3. 用 USB 開機，然後從 UNetbootin 開機選單中選取 ESXi-<version>-standard installer。

    4. 過程中如果有遇到狀況，會看到紫色的畫面，否則就一路來到 "Welcome to the VMware ESXi 5.1.0 Installation" 的畫面。

    5. 選擇要將安裝哪個磁碟，如果該磁碟之前已經裝有 ESXi，會提示重裝還是昇級（可保留 VMFS datastore）。

    6. 設定 root 帳號的密碼，至少要 7 個字元。

    7. 安裝完成後會提示兩個很重要的訊息：

  • 一開始會有 60 天的 evaluation period，之後要註冊 license key 才行（下載頁面有提供）。

  • 可以透過 vSphere Client 或 Direct Control User Interface (DCUI) 管理。

    1. 用 vSphere Client 連到 ESXi，切到 Configuration > Software > Licensed Features > 右上方的 Edit… 就可以輸入下載頁面提示的 license key。
