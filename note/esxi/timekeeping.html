ESXi / Timekeeping

按 Timekeeping in VMware Virtual Machines 的說法，存在所謂的 Virtual CMOS RTC (Real Time Clock)。從 guest OS 修改時間時，就算工具聲稱會寫回 hardware clock （例如 Linux 的 hwclock），其實動到的也只是 Virtual CMOS RTC 而已，因此不用擔心同一個 VM host 下不同 guest OS 的時間會相互干擾的問題。

要調整 Virtual CMOS 的時間，最簡單直覺的方式就是直接從 VM 的 BIOS 畫面下去調整。
