Pip / 查詢已安裝或可用的套件

查詢可安裝的套件可以利用 pip search，基本的用法如下：（完整的用法可以用 pip help search 取得）

pip search <keyword>

以 RIDE 為例：

$ pip search robotframework
robotframework-ioslibrary - Robot Framework Automation Library for iOS 
robotframework            - A generic test automation framework
  INSTALLED: 2.7.1 
  LATEST:    2.7.5
robotframework-ride       - RIDE :: Robot Framework Test Data Editor
...
只要套件名稱或簡介有出現關鍵字，都會被列出來。
如果某個套件已安裝，會一併顯示 "已安裝的版號" 跟 "最新的版號"，因此 pip search 也可以用來查詢已安裝套件的狀態。

除了 pip search 之外，pip freeze 也可以看到安裝的版號，例如：

$ pip freeze | grep robotframework
robotframework==2.7.1
robotframework-seleniumlibrary==2.9
