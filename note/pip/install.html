Pip / 安裝

如果已經裝有 EasyInstall，最簡單的方式就是透過它來安裝 pip 套件：

$ easy_install pip

如果沒有的話，也只要下載 get-pip.py，然後直接執行它即可：

$ python get-pip.py

安裝完成後，用 pip --verison 檢查一下安裝。
