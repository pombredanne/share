:title: [Fiddler] 安裝 Fiddler

Fiddler 的版本目前有兩個分支－Fiddler 2.x 跟 4.x，前者需要用到 .NET Framework 2.x，後者則用到 .NET Framework 4.x。

TIP: 建議搭配系統內建的 .NET Framework，例如 Windows 7 內建 .NET Framework 3.5，建議安裝 Fiddler 2.x，如果是 Windows 8 的話，由於內建 .NET Framework 4.0，建議安裝 Fiddler 4.x。
