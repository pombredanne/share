Python / Files / I/O

讀寫檔案之前要先開啟，通常都是用 open() 來開啟檔案，然後會得到一個 file 物件。open() 的用法如下：

open(name[, mode])

其中 name 是 file name，可以是相對或絕對的表示法（相對於 CWD），而 mode 指的是 mode string，它關係到：

  • 可否寫入？

    唯讀（r）、覆寫（w） 或附加（a）。

  • 採用 text mode 或 binary mode？

    open() 預設採用 text mode，適合用在文字檔的讀寫，因為過程中會自動做換行字元的轉換。由於這樣的轉換，二進位檔的讀寫則應該要改用 binary mode (b)，否則檔案內容可能會遭到破壞。

常見的 mode string 有：（下面的 U 代表 universal newline support，可以參考這裡的說明）

  • rU - 讀取文字檔。

  • w - 寫入文字檔。

  • rb - 讀取二進位檔或文字檔（不做換行字元的轉換）。

  • wb - 寫入二進位檔或文字檔（不做換行字元的轉換）。
