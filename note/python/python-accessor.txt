= Python Programming: 邪惡的 getter/setter？ =

 * 即便沒有對應的 setXXX，但 getXXX 暗示後面有計算的動作，也暗示著這次呼叫跟下次可能會有不同的結果。參考 file-like object 的設計
 * 要用 getter/setter 或 attribute/property 時，通常都是 data 的載體。
