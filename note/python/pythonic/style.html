Pythonic / Code Style/Conventions

#!/usr/bin/env python 
"""This program (or module) ... 
...
"""
import os 
import mymodule

__authors__ = ['Jeremy Kao', 'Joanne Liu'] 
__author__  = ','.join(__authors__)        

__maintainer__ = 'Jeremy Kao'              
__email__ = 'imsardine@gmail.com'

...
如果這支 .py 可能被當做 main script 使用時，要加上這一行。
接下來是 module docstring，第一行是 summary。
import 必須接在 docstring 後。先是 built-in module、third-party module 然後才是自己的 module。
Authorship information。
多個 author 時用逗號分開，通常用 __authors__ 自動建立。
Maintanence information。維護的人通常只有一個，連絡的 email 裡 __email__ 來訂義。
