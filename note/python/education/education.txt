 * http://www.pythontutor.com/[Online Python Tutor] - 用圖形說明每一行 code 的執行，酷到不行！

 * http://docs.python.org/library/turtle.html[Turtle graphics for Tk — Python v2.6.2 documentation]
 * http://code.google.com/p/crunchy/[Crunchy] - 看不太懂在做什麼, 但它的作者與 RUR-PLE 是同一個人 - André Roberge
