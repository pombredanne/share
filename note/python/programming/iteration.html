Python / Programming / Iteration - Iterable & Iterator

如果遶行的時候也需要一個 counter，傳統的做法是自己維護一個 counter：

>>> idx = 0
>>> letters = ['a', 'b', 'c']
>>> for letter in letters:
...     print idx, letter,
...     idx += 1
...
0 a 1 b 2 c

但 Python 提供了一個 enumerate()，可以在遶行的同時得知取出的順序：

enumerate(sequence, start=0)
>>> for idx, letter in enumerate(letters): 
...     print idx, letter,
...
0 a 1 b 2 c
>>>
>>>
>>> for idx, letter in enumerate(letters, start=1): 
...     print idx, letter,
...
1 a 2 b 3 c
enumerate() 會回傳一個 iterator，逐項取出 2-tuple，第一個項目是 counter 的內容。
Counter 的起始值可以用 start 來設定。
