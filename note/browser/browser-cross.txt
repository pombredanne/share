= Cross-Browser Compatibility =

同一份網頁經過不同 browser 解讀，最後呈現出來的結果多少會有些差異（最明顯的就是 form element 的外觀很不一樣）。除了畫面上的表現之外，也要注意 JavaScript 或 Ajax 的處理方式可能不同。
