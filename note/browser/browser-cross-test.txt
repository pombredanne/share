= Cross-Browser Compatibility Testing =

目的在測試 Web Application 是否有正確地處理 link:browser-cross.html[Cross-Browser] 的議題，也就是在不同平台（Browser + OS）上畫面都能正確地呈現，功能也都正常。

== 工具 ==

 * http://browsershots.org/[Check Browser Compatibility, Cross Platform Browser Test - Browsershots] - 由分散在全世界各地的使用者用他們的 browser 瀏覽特定的網址後，再將畫面的截圖傳回匯整，可以看到同一個網頁在不同 browser 上的表現。
 * http://www.browserstack.com/[Cross Browser Testing Tool - BrowserStack]
