= Watir - Web Application Testing in Ruby =

Watir 唸作 "water"，是一個用來做 link:../web/web-test-automation.html[web automation] 的 Ruby library。其他語言的實作有：

 * http://watij.com/[Watij] (Java)
 * http://watin.org/[WatiN] (.Net)

Watir 一開始只支援 Windows/IE，下一代的 link:watir-webdriver.html[Watir-WebDriver] 就沒有這個限制。
