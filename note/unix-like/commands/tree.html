Unix-like / Commands / tree

tree - list contents of directories in a tree-like format.

tree [<options>] [<directory> ...]

以樹狀結構印出 directory（或 current directory）的內容。常見的用法有：

  • tree -apugF [-L <level>] [-l] [<directory>]

    查看 directory 的內容。其中 -L <level> 可以控制最多只展開 level 層目錄，加上 -l 則可以展開（follow）指向其他目錄的 symlink。

  • tree -da [<directory>]

    查看 diretory 的目錄結構。

常用的 options 有：

-a

一併印出隱藏檔。

-d

只印出目錄。

-l

遇到指向其他目錄的 symlink 時，不只印出 target，也繼續展開（follow）內容。

-L <level>

最多只展開 level 層目錄。其中 level 的最小值是 1，表示不展開目錄。

-p

額外印出 mode bits。

-u

額外印出 owner 的名稱或 UID。

-g

額外印出 group 的名稱或 GID。

-F

在檔名後面用不同的符號標示不同性質的檔案，例如 / 表示目錄、* 表示執行檔等。

-D

額外印出 mtime。

--inodes

額外印出 inode number。

--noreport

結尾處不要印出檔案跟目錄的數量。

--dirsfirst

優先印出目錄，然後才是其他檔案。
