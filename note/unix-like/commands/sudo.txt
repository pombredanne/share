:title: [Unix-like] sudo

 sudo - execute a command as another user

`sudo` 讓你能以另一個使用者的權限（預設是 root）執行指令。下面的指令可以切換成 root 身份：

 sudo -i

 $ sudo [-u <user_name>] <command>

如果 invoking user 是 root，或者跟 target user 是同一個人，就不會被提示要輸入密碼。
