`service` - run a System V init script

`service` 可以用來執行 `/etc/init.d/` 底下的 init scripts，語法是：

 service SCRIPT COMMAND [OPTIONS]

其中 'SCRIPT' 對應 `/etc/init.d/SCRIPT`，而 'COMMAND' 跟 'OPTIONS' 則會原封不動地傳給 init script。

每個 init script 都至少要支援兩個 command - `start` 跟 `stop`，另外有個特殊的 command `--full-restart`，會連續呼叫 init script，分別傳 `start` 跟 `stop` 進去。（通常 init script 會實作 `restart` 這個 command，但看來這並不會標準？）
