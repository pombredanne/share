[Sikuli] Mac 上的 Sikuli

  1. 確認 JRE 已經安裝。

  2. 下載 Sikuli-X-<version>-oxs-10.6.dmg 後開啟，然後將 Sikuli-IDE 拖進 /Applications 即可，結構如下：

    /Applications/Sikuli-IDE.app
    |-- Contents/
|   |-- Frameworks/
|   |-- MacOS/
|   |-- PkgInfo/
|   |-- Resources/
|   |   |-- Java/
|   |   |   |-- sikuli-script.jar
|   |   |   `-- *.jar
|   |   |-- SikuliIde.icns
|   |   `-- SikuliSrc.icns
|   `-- Info.plist
`-- sikuli-ide.sh

  3. 從 Launchpad 就可以找到 Sikuli-IDE。

在 OS X 10.8 上測過，給 OS X 10.6/10.7 用的 Sikuli X 1.0 rc3 (r930) 也可以執行。

Dock 上圖示的大小會因為同時間出現在 Dock 裡圖示的數量影響，如果要從 Dock 帶出某個應用程式，很可能因為圖示大小不同而失敗，建議改用 openApp()。
