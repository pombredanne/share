:title: Sikuli Java API

[quote,'http://code.google.com/p/sikuli-api/[sikuli-api - Sikuli Java API]']
________________________________________________________________________________
Sikuli API for Java provides image-based GUI automation functionalities to Java programmers. It is created and will be actively maintained by Sikuli Lab. The effort of making this API is a response to the desire of many Sikuli users to use Sikuli Script's functionalities directly in their Java programs, rather than writing Jython scripts. This new Java library has a *re-designed API* and includes several new functions that were *not available in the original Sikuli Script*, such as the abilities to match colors, handle events, and find geometric patterns such as rectangular buttons.
________________________________________________________________________________

Sikuli Java API（簡稱 Sikuli API） 在 2012-10-20 釋出 https://bitbucket.org/doubleshow/sikuli-api/commits/all/tip/sikuli-api-1.0.0[1.0.0] 版，主要由 http://www.tomyeh.info/[Tom Yeh] 負責開發。

Sikuli API 是全新設計的 API，用法與 http://doc.sikuli.org/faq/030-java-dev.html[How to use Sikuli Script in your JAVA programs] 提到的方式大不相同，而且有許多新功能是 Sikuli Script 所沒有的。
