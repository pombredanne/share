RIDE / Keyboard Shortcuts

熟記一些常用的快捷鍵，可以加快在 RIDE 裡編寫 test case 的速度，減少手指頭在鍵盤跟滑鼠之間來回移動的次數。

  • F5 - 開啟 Search Keywords 的對話框。

  • F6 - 預覽 test data 轉換成 HTML、space separated 或 pipe separated 的樣子，按 Print 可以開始列印。

  • F2 - 更改名稱，而且也會自動調整其他有用到這個名稱的地方。

  • Alt + Z/X - RIDE 會記錄左側樹狀節點被選取的歷程，Alt + Z/X 可以快速地在其間做切換，當左側的樹狀結構變得複雜時尤其有幫助。

  • Ctrl/Alt + Up/Down - 調整上下位置。大部份情況下都是用 Ctrl，目前只有 test case 裡 step 的上下位置要用 Alt 來做調整。

  • Ctrl + Shift + C - 複製 test case。

  • Ctrl + I - 在前面插入一行。

  • Ctrl + D - 刪除所在行或選取的多個行。

  • Ctrl + B - 查看 user keyword 的內容。可以搭配 Alt + Z 回到原來的位置。
