[RIDE] 安裝／昇級

RIDE 目前 (2012-04-17) 最新版是 RIDE 0.43.3（仍處於 alpha），只能跑在 Python 2.5+ 的環境（不支援 Python 3 或其他 Jython、Iron Python 等）。

安裝 RIDE 之前，要先完成 wxPython (with Unicode support) 的安裝。Windows 必須透過 binary 來安裝（不能透過 easy install 來裝，大概是因為有 native 的東西吧？），在 Ubuntu 下則只要安裝 python-wxgtk2.8 套件即可。另外從 0.36 開始就直接內建（相依？） Robot Framework，所以不用另外安裝。

用 pip 或 easy_install 安裝 robotframework-ride 套件即可。

$ pip install robot-framework-ride
$ easy_install robotframework-ride

不同平台的安裝方式有些差異：

  • 在 Windows 上安裝 RIDE

  • 在 Mac 上安裝 RIDE

  • 在 Linux 上安裝 RIDE
