PDF Diff

最簡單的方式就是透過 pdftotext 之類的工具先將 PDF 轉成文字，再用純文字的比對工具比對檔案間的差異。

$ sudo apt-get install diffpdf
$ diffpdf [file1] [file2]

DiffPDF 比較可惜的地方是無法略過 comments 的比較，不過 comparison mode 選 Text 可以有類似的效果。
