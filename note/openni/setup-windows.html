OpenNI / Setup (Windows)

  1. 從官網下載 OpenNI SDK、OpenNI-Compliant Sensor Driver 跟 NiTE。

    2012-12 OpenNI 2.x 釋出之後，官方將 OpenNI 1.x 的下載位置移到這裡，版號分別是：

    • OpenNI SDK v1.5.4.0

    • OpenNI-Compliant Sensor Driver v5.1.2.1

    • NiTE v1.5.2.21

    由於 OpenNI-Compliant Sensor Driver（也就是 SensorKinect）官網提供的版本目前（2012-12）只支援 Kinect for Xbox，並不相容於 Kinect for Windows，要到avin2/SensorKinect 的 unstable 分支下載最新的版本才行。

  2. 安裝 OpenNI SDK，預設會裝在 %ProgramFiles%\OpenNI。

  3. OpenNI SDK 帶有公版的驅動程式，支援 ASUS Xtion，其餘則要加裝對應的驅動程式。

    • Kinect for Xbox - 安裝 OpenNI-Compliant Sensor Driver，預設會裝在 %ProgramFiles%\PrimeSense\Driver。

    • Kinect for Windows

      • 安裝 unstable 分支的 OpenNI-Compliant Sensor Driver 或，

      • 安裝 kinect-mssdk-openni-bridge。

  4. 安裝 NiTE，預設會裝在 %ProgramFiles%\PrimeSense\NITE。

  5. 最後執行一些 OpenNI 或 NiTE 所附的範例程式（例如 NiViewer 或 Sample-StickFigure），確認 OpenNI 可以正常驅動硬體裝置。
