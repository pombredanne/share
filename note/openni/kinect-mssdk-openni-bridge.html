OpenNI / kinect-mssdk-openni-bridg

由於 SensorKinect 只支援 Kinect for Xbox，而不相容於 Kinect for Windows，所以作者開發了一個 OpenNI 的 plug-in，做為 OpenNI 跟 Kinect SDK 之間的 bridge。程式一樣呼叫 OpenNI API，這個 bridge 會轉呼由 Kinect SDK 來進行處理。

採用 bridge 的做法將 OpenNI 跟 Kinect SDK 串起來，不必犠牲 near mode 的功能，也支援 audio。這或許也是為什麼 OpenNI 2 也是間接透過 Kinect SDK 對 Kinect for Windows 支援的考量…

安裝的步驟：

  1. 安裝好 Kinect SDK 跟 OpenNI SDK。

    由於 kinect-mssdk-openni-bridge 目前只提供 32-bit 的版本，所以 OpenNI SDK 也要裝 32-bit

  2. 從官網下載 kinect-mssdk-openni-bridge-v<version>-for-1.5.2.23.zip。

    其中 "for-1.5.2.23" 指的是編譯時所採用的 OpenNI SDK 版本，只要採用的 OpenNI SDK 版本相近即可。

  3. 解開壓縮檔執行 install.bat，就會呼叫 OpenNI 的 niReg 為 bin\kinect-mssdk-openni-bridge-V10Release.dll 進行註冊。

  4. 最後執行一些 OpenNI 所附的範例程式（例如 NiViewer），確認 OpenNI 跟 Kinect SDK 有成功地被串起來。
