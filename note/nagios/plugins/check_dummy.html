Nagios / Plugins / Official / check_dummy

check_dummy <return_code> [<text_output>]

check_dummy 完全符合 plugin API 的兩項要求，只是它內部什麼檢查都沒做，只是傳回指定的 return code，並印出 text_output 而已；如果 text_output 沒有給的話，預設印出不同 return code 所代表的 state，例如 1 會印出 "WARNING"。

$ ./check_dummy 1
WARNING
$ ./check_dummy 3 "Something is in trouble"
UNKNOWN: Something is in trouble
check_dummy 在實務上可以做為一個 placeholder，讓整個 host/service check 可以完整地串起來，待真正的 check command 準備好之後，再置換掉即可。
