Robotium RC / Architecture

組成：

  • Remote Controller

  • Robotium Test Runner

  • SAFS TCP Messenger - AUT-agnostic 只要安裝一次

在 SAFS TCP Messenger 兩側加上 Embedded Robotium Solo 跟 Robotium RC Solo 看似 Robotium Test Runner 執行起來時（透過 instrumentation），會直接把 SAFS TCP Messenger 帶起來。 之間溝通的橋樑，它也不清楚之間溝通的細節，只要符合 protocol 就好。
