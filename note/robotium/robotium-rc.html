Robotium / Remote Control (RC)

Robotium RC 是很重要的一步，做為一個 testing tool，有很大的機會要被整合進 testing framework，如果執行的單位不再是不可切割的 JUnit test…

Robotium RC 喊了很久，但一直沒有明顯的進展，直到：

  • 2012-06-20 Groupon 發表 RoboRemote。

  • 2012-07-31 SAFS 發表 Robotium Remote Control
