 * 由 Brad Cox 發明。在 C 加上 OO（仿 SamllTak）的擴充，所以 C 是 Objective-C 的子集（形容自己為覆蓋於 C 語言上的一層薄紗），所有 C 程式都可以用 Objective-C 編譯器編譯。
 * gcc 支持 Objective-C，所以只要有 gcc 就可以編譯 Objective-C。
 * 主要應用於 Mac OS X 跟 iOS 這兩個衍生自 NeXTSTEP 的系統，而 Objective-C 是 NeXTSTEP 的基本語言。
 * NeXT 獲得 Objective-C 的授權，開發出 NEXTSTEP 的開發環境（這就是為什麼 Objective-C 跟 NEXTSTEP 會有關聯）。之後 NeXT 和 Sun 共同提出了 OPENSTEP 的標準（GNUStep 就是 GNU 的實作)。之後 Apple 買下了 NeXT 這家公司，OPENSTEP/NEXTSTEP 成為 OS X 的基礎，將它稱做 Cocoa。
