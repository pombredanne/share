= SAFS (Software Automation Framework Support) =

SAFS 是一套跨平台的 keyword-driven testing framework，整合不同 testing tools 並提供許多 keyword libraries，可以用來測試 Java、Web/Html、 Flex、.Net、WPF、Windows、link:ios-testing.html[iOS] 跟 link:android-testing.html[Android] 不同形式的應用程式。

另外強調可以做 language localization testing (NLS) 跟 image-based testing，不知是怎麼辦到的...
