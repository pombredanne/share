== 參考資料 ==

 * http://pcsupport.about.com/od/windows-8/a/elevated-command-prompt-windows-8.htm[How To Open an Elevated Command Prompt in Windows 8] - 按 Win + Q 找到 Command Prompt 後，點右鍵下方就會出現 Run as administrator 的選項。另外從 Task Manager 裡，File > Run new task 跟 Win + R 開出來的 Run 視窗很像，但下方會多出 "Create this task with administrative privileges." 的選項，更為方便。
