:title: [Windows 8] Mouseless

Windows 8 用滑鼠操作起來不是很方便，還好很多都有對應的鍵盤組合鍵可以替代。常用的有：

TIP: 在 VirtualBox for Mac 下，Windows Logo 鍵可以用空白鍵右側的 Command 替代。

 * Win - 可以來回在 Desktop 跟 Start Screen 間切換。
 * Win + C - 叫出右側的 charms，上下鍵可以選取 Search/Share/Start/Devices/Settings，按 Esc 可以離開。
 * Win + Q/W/F - 搜尋 Apps、Settings 或 Files。
 * Win + Z - 叫出 context menu，接著透過 Tab 或 Shift + Tab 在不同的項目間移動。
 * Win + D - 顯示桌面。
 * Win + Tab - 在最近常用的應用程式間切換，跟 Alt + Tab 不太一樣。
