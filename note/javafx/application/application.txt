:title: [JavaFX] Application

 * entry point - Main class extends `javafx.application.Application`
 * instantiate main class -> `Application.init()` -> `Application.start(javafx.stage.Stage)`
 * `Platform.exit()` 或 last windows 關掉（`Platform.implicitExit == true`）
 * `Application.stop()`
