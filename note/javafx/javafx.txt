= JavaFX - The Rich Client Platform =

 * Sun Microsystems, JavaOne 2007
 * client-rich applications (RIA)
   ** development, replace complex Java Swing and Java 2D
   ** deployment on consumer devices (cell phones, televisions, in-dash car systems, browsers)
 * JavaOne 2011 -> JavaFX 2.0, deprecate the JavaFX Script language in favor of using Java and the JavaFX 2.0 API
