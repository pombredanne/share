JavaFX / Recipes / Embedding Google Maps

透過 JavaScript API 可以將 Google Maps 內嵌到網頁裡，而 JavaFX 可以透過 WebView 來顯示網頁的內容。透過這兩者的組合，就可以將 Google Maps 嵌入 JavaFX。

延伸閱讀

  • Google Maps JavaScript API

  • WebView

參考資料

網站

  • V3: The Solution for Maps Applications for both the Desktop and Mobile Devices - Google Maps JavaScript API v3 — Google Developers

文件

  • Steve On Java - JavaFX » JavaFX and HTML5 at JavaOne India 閱讀 (2012-05-06) - 示範了內嵌 Google Maps (content.html) 的做法，而且可以從 JavaFX 切換不同的地點。

  • Java-Buddy: Embed Google Maps in JavaFX WebView 閱讀 (2012-03-28) - 透過 WebView 將 Google Maps 嵌進 JavaFX。有人提到滑鼠拖曳、滾輪等都沒有作用…

  • Maps in JavaFX 2.0 // JavaFX News, Demos and Insight // FX Experience 閱讀 (2011-05-31) - 內嵌 Google/Yahoo/Bing Maps。但在 JavaFX 2.2.4 下程式要做些調整（TextBox → TextField，stage.setVisible(true) → stage.show()…），但執行起來馬上就 crash 了… 不過 JavaFX 透過 JavaScript 間接控制 Google Maps 的部份還是很值得參考。

其他文件

  • Issue 4458 - gmaps-api-issues - Hello World example broken in Chrome 22 - Google Maps API bug reports and feature requests - Google Project Hosting (2012-09-26) - 雖然官方問說什麼是 JavaFX？後來有人回報說 JavaFX 2.2 上沒問題了。

  • java - embedding Google Maps in a JavaFX 2.2 desktop application throwing exception and generaly not performing - Stack Overflow 閱讀 (2012-09-12) - 提到昇級至 7u7 之後 gesture 就沒有作用了？由於 JavaFX 2.2 開始有實作 touch events，所以這是 WebView 的問題。在問題解決之前，建議先採用 JavaFX 2.1，這是沒有實作 touch event 但又支援 JavaScript to JavaFX bridge 的版本。

  • Issue 4238 - gmaps-api-issues - Bug: Panning with mouse does not work if touch events are enabled - Google Maps API bug reports and feature requests - Google Project Hosting 閱讀 (2012-06-18) - Google Maps API v3 在 enable touch 時，滑鼠無法拖曳地圖，但 v2 跟 Yahoo/Bing 都沒有這個問題。
