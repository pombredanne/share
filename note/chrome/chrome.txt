== 雜記 ==

 * Release cycle 目前是 6 weeks，跟 Firefox 一樣。
 * 透過 Ubuntu 的 package manager 只能裝到 Chromium，要裝 Chrome 的話，還是得到官網去。
 * Chrome Sync 可以在多台電腦同步 Chrome 所有的設定。如果發現沒有同步，可能是因為常駐在工具列的 Chrome 沒有關閉所造成。
