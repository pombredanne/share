:title: [Chrome] Mouseless

Chrome 內建的 keyboard shortcuts 有：

 * Ctrl + N	- 開一個新的視窗。
 * Ctrl + T - 開一個新的頁籤。
 * Ctrl + L - 移動到網址列並全選。
 * Ctrl + K / E - 移動到網址列並預先輸入一個 `?`，輸入關鍵字，按下 Enter 就可以用 default search engine 進行搜尋。
 * Alt+D - 將網址列的內容全選，跟 IE 或 Firefox 一樣的。

相關的 extensions 有：

 * https://chrome.google.com/webstore/detail/deadmouse/kioijmpindokaaahaeigkkkbogccljhm[DeadMouse] - 直接輸入 anchor 的文字，符合條件的連結就會扭動（按 Escape 離開），按下 Enter 開啟連結，或是 Shift + Enter 開在新 tab，有多個 anchor 符合條件時，可以用 Tab 或 Shift + Tab 來回移動。不適合用在 Twitter 或 Gmail 之類提供有 keyboard shortcuts 的網站。
