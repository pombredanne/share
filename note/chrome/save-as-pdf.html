Chrome Save As PDF

調整紙張大小

Chrome 的列印預覽畫面不能調整紙張大小，但會受 CSS 裡的設定所影響。

@media print {
  body {
    margin: 0;
    padding: 0;
  }
  @page {
    margin: 0;
    size: 11in 8.5in; 
  }
}
透過 size 可以調整紙張大小。例如 size: 5.83in 8.27in; (A5)、size: 4.13in 5.83in; (A6) 可印出適合在 Kindle Touch 上閱讀的大小。
