Android: 下載 APK

可能的方案

目前（2012-06-30）有兩個工具可以用來從 Google Play 上直接取得 APK 檔：

  • Real APK Leecher | Torkaland - 免安裝的 Java 應用程式，設定上較簡單。

  • APK Downloader - Chrome 的外掛，，不過 Chrome 要以特殊的方式啟動。

這兩個工具要設定的資訊一樣：Email、Password 跟 Device ID。其中 Device ID 可以透過手機撥號程式輸入 *#*#8255#*#* 叫出 GTalk Service Monitor 來查看（這個方法也適用在 Emulator 上）
