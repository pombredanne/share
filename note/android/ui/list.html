[Android] List

ListView 可以將多個資料項以垂直捲動的清單表現出來。

ListView 透過 adapter 來取得資料，可以搭配不同的 adapter 跟不同的資料來源（data set）連接。最簡單的用法就是搭配 ArrayAdapter，直接將 array 或 List 做為資料來源。

ArraryAdapter 間接實作了 Adapter 介面。Adapter 居於畫面上的呈現（AdapterView；ListView 就是 AdapterView 的一種）跟背後的 data set 之間，除了每個順位（position）的 data item 之外，也決定了 data item 要怎麼呈現（View）。
ListArrayActivity.java
public class ListArrayActivity extends Activity {

    private static final String[] DATA_ITEMS = new String[] { "Windows", "Mac OS X", "Linux" };

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.list_array_activity);

        ListView list = (ListView)findViewById(R.id.list);
        list.setAdapter(new ArrayAdapter<String>(
            this,
            android.R.layout.simple_list_item_1, 
            android.R.id.text1,                  
            DATA_ITEMS));
        ));
    }

}
