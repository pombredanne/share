Android Overlays/Skins

Android 製造商可以在 OS 之上訂制自己的 UI 介面，稱之為 Manufacturer Overlay 或 Skin，不只是畫面呈現的方式不同，內建的應用程式也可能被置換成客製的版本，使得各家手機操作起來有截然不同的用戶體驗（UX）。

其中比較知名的有：

  • Motorola Motoblur

  • Samsung TouchWiz

  • HTC Sense
