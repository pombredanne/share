[Android][ADB] Shell Commands

除了 built-in commands，例如 cd、pwd、export、set、type 等，所有的 shell command 都在 /system/bin 底下，有上百個之多。

$ adb shell
$ echo $PATH
/sbin:/vendor/bin:/system/sbin:/system/bin:/system/xbin
