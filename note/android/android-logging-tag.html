[Android] Log Tag

Tag 可以用來識別 log message 是從哪裡來的，

通常會在 class 定義一個 TAG 的常數，重複用在 Log.x() 的呼叫：

private static final String TAG = "MyActivity";
private static final String TAG = XXX.class.getSimpleName();
