Command Line

開發上不會因為沒有使用 Eclipse ADT 而少了某些支援，因為 ADT 也是包裝 Android SDK 所提供的功能而已。

android create project \
    --package com.example.helloandroid \
    --activity HelloAndroid \
    --target 2 \
    --path <path-to-your-project>/HelloAndroid
