[Android] View Server

Hierarchy Viewer 只能搭配特定版本的 Android 系統運作，大概是因為這些特殊版本的系統才搭載有 View Server 吧？

有沒有可能讓 View Server 執行在一般手機裡呢？

  • Commands available for View Server class - Google Groups - 有人提到 The ViewServer is not a public API and should NEVER be used as such. It also won’t be enabled on production devices (phones sold in stores, for instance.) Do not rely on it. It’s also not designed for real time interaction and can be very slow. The protocol can also change without warning in any version of Android.
