= 透過 ADB 安裝／移除套件 =

如果手邊有 APK 檔，透過 `install` 這個 device command 就可以進行安裝：

 $ adb shell <apk_file>

--------------------------------------------------------------------------------
$ adb install Development.apk 
2087 KB/s (112988 bytes in 0.052s)
	pkg: /data/local/tmp/Development.apk
Success
$ echo $?
0
$
$ adb install Development.apk 
2087 KB/s (112988 bytes in 0.052s)
	pkg: /data/local/tmp/Development.apk
Failure [INSTALL_FAILED_ALREADY_EXISTS] <1>
$ echo $?
0
--------------------------------------------------------------------------------
<1> 重覆安裝同一個套件會失敗（就算是版本新一點，也不會有 "昇級" 的效果）。雖然 'Failure [...]' 寫著失敗的原因，但並沒有在 exit status 反應出來。

透過 link:android-adb-activity-manager.html[Activity Manager] 可以直接叫出 Google Play 要安裝某個套件的畫面。以 https://play.google.com/store/apps/details?id=org.jtb.alogcat[aLogcat] (`org.jtb.alogcat`) 為例：

--------------------------------------------------------------------------------
$ am start market://details?id=org.jtb.alogcat
Starting: Intent { act=android.intent.action.VIEW dat=market://details?id=org.jtb.alogcat } <1>
--------------------------------------------------------------------------------

要移除套件的話，

 $ adb uninstall <package_name>

--------------------------------------------------------
$ adb uninstall com.android.development
Success
$ adb uninstall com.android.development
Failure
--------------------------------------------------------

在這個過程中，都可以用 link:android-adb-package-manager.html[Package Manager] 先確認一下該套件是否已經安裝或移除了：

--------------------------------------------------------------------------------
$ pm list packages
package:android
package:android.tts
package:at.abraxas.amarino
...
--------------------------------------------------------------------------------
