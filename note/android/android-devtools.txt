= Android Dev Tools =

從 Emulator 將 APK 拿出來：

 $ adb -e pull /system/app/Development.apk /tmp

將 APK 裝到實際的裝置：

 $ adb -d install /tmp/Development.apk

如果之前有安裝過其他版本，必須要先移除才行：

 $ adb -d uninstall com.android.development
