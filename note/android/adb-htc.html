Android / ADB / Hardwares / HTC

HTC 要到官網支援服務下找特定機型，下載 HTC Sync。

HTC Sync 爆肥（高達 153 MB），在 XP SP2 上安裝時，提示要先安裝一大堆的東西：

  • MSXML 4.0 SP3

  • Windows Media Format 11 Runtime

  • Microsoft Visual C++ 2008 SP1 Redistributable Package (x86)

  • Microsoft .NET Framework 2.0 SP2

  • Adobe AIR 3.2.0.2070 （tray icon 點開的畫面是跑 AIR）

  • HTC BMP USB Driver 1.0.5375 (32-bit)

  • HTC Driver Installer 3.0.0.021

安裝過後，在 Control Panel > Programs 下會看到 3 個項目：

  • HTC BMP USB Driver

  • HTC Driver Installer

  • HTC Sync

2012-05-17 安裝 HTC Sync 3.2.20 的過程中，走到 HTC Driver Installer 時，可以按 Cancel 暫停安裝的動作，這時候就可以在 Local Settings\Temp\ 下找到 HTC BMP USB Driver x86.msi 跟 HTCDriver.exe。實驗發現，只要安裝 HTCDriver.exe 就可以透過 ADB 連接上，BMP USB Driver 不用。
