:title: [Bash] Arguments

 - `$0` 是 script 本身
 - `$1` ... `$n`
 - `$#` 可以取出參數的數量，不含 `$0` 本身。

--------------------------------------------------------------------------------
if [ $# -ne 2 ]; then
    echo "Usage: $0 arg1"
    exit -1
fi
--------------------------------------------------------------------------------
