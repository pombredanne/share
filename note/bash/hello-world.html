腦袋不是用來裝東西，而是用來思考問題的；所以我把懶得記的、記不住的，通通寫在這裡...

Bash / Hello, World!

hello_bash.sh
#!/bin/bash           

# Greeting in Bash    
echo Hello Bash!
sleep 5s              
exit 0 # success
Bash 的 shebang。
# 以後的部份都會被視為註解。除了可以擺在一行的最前面，也可以擺在某行指令的後面。
暫停 5 秒鐘。
可以用 exit 回傳 exit status （否則預設回傳最後一個 command 的 exit status）。

執行結果如下：

$ ln -s hello_bash.sh hello_sh
$ chmod a+x hello_bash 
$ ./hello_bash
Hello Bash!
$ echo $? 
0
把它變成 executable。
印出 ./hello_bash 的 exit status。
