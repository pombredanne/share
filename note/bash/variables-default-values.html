Bash: 指派預設值給沒有定義的變數

延伸閱讀

  • Parameter Expansion 

$ name=jeremy
$ echo ${name:-anonymous}
jeremy
$ unset name
$ echo ${name:-anonymous} 
anonymous
$ name=
$ echo ${name:-anonymous} 
anonymous
變數沒有定義時，預設值就會生效。
變數的內容不是空白時才算有定義。
