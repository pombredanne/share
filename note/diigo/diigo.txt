= Diigo - Collect and Highlight, Then Remember =

Diigo 提供了很多工具，可以用來提昇自己網站跟 Diigo 之間的黏性：

 * link:linkrolls.html[Linkrolls]
 * link:add-to-diigo.html[Add to Diigo]

－些使用上的經驗：

 * 閱讀過後（加註 highlights 跟 comments），別忘了在記憶最鮮明的時候，將對該份文件整體的心得寫在 bookmark 的 description 裡。雖然日後還是可以從 annotations "慢慢" 找回當初的感覺，但如果當時有把它寫下來，就可以一眼看出文件的重點。

You can upload a cached page while bookmarking using the Diigo toolbar.

This is best for important or temporary pages that you really want to preserve, and also for pages that are behind password. PDF and other file formats can also be uploaded.

Once submitted, you should see the cached page in your library within seconds to a minute. You can upload cached page of the same url at different times if you are interested in how the page changes over time. 可以比對差異嗎？

Highlight::

 * 直接做 highlight，也會成為一個 bookmark。建議先做 bookmark

Sticky Notes::

 * public/private 指的是其他 Diigo 的會員是否看得到，如果將 annotated link 分享出去，並不會因設定成 private 而看不到。

一些對 Diigo 的看法：

 * Research -> 蒐集書籤、畫重點（用不同顏色的螢光筆）、貼便條紙、彙整、分享在網路上所找到的資訊 -> My Library -> 將找到的資訊與重點分享給同組的組員
 * 相對於用 PDF 做 annotation，缺點就是 web annotation 需要有網路時才能作用，沒法做 diff
 * 萬一哪些 Diigo 掛點，annotated link 背後真正的連結就不見了，更慘！
