VMware Workstation

Unity

透過這項功能（View > Unity），可以將 guest OS 裡的應用程式無縫地顯示在 host OS 上。跟 VirtualBox 的 Seamless Mode 很像，但兩者的表現方式不太一樣。

滑鼠移到 Windows 的 "開始" 時，會浮現 guest OS 的選單。
