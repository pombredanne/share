Base64 / Python

Python 裡 Base64 編／解碼的功能由 base64 這個 module 提供，最標準的用法是：

  • standard_b64encode(s)

  • standard_b64decode(s) 

>>> import base64
>>> base64.standard_b64encode('abca')
'YWJjYQ=='
>>> base64.standard_b64decode('YWJjYQ==')
'abca'

為什麼會說 "最標準“ 的用法呢？因為 Base64 用來表示編碼結果的 64 個字元，最後更個字元（+ 跟 /）在某些場合下可能不適用，可以代換成不同的字元，只要解碼時套用相同的規則即可。

  • b64encode(s[, altchars])

  • b64decode(s[, altchars])
