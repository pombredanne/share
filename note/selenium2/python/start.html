Selenium 2/WebDriver / API / Python / Getting Started

用 pip 安裝 selenium 套件：

$ pip install selenium
>>> from selenium import webdriver
>>> dir(webdriver)
['ActionChains', 'Chrome', 'ChromeOptions', 'DesiredCapabilities', 'Firefox', 'FirefoxProfile', 'Ie', 'Opera', 'PhantomJS', 'Proxy', 'Remote', 'TouchActions', '__builtins__', '__doc__', '__file__', '__name__', '__package__', '__path__', '__version__', 'chrome', 'common', 'firefox', 'ie', 'opera', 'phantomjs', 'remote']

這裡拿 Firefox Driver 簡單做個測試：

>>> from selenium import webdriver
>>> from selenium.webdriver.common.keys import Keys
>>> driver = webdriver.Firefox() # 
>>> type(driver)
<class 'selenium.webdriver.firefox.webdriver.WebDriver'>
>>>
>>> driver.get('http://en.wikipedia.org') # 
>>> search_element = driver.find_element_by_name('search')          # 
>>> type(search_element)
<class 'selenium.webdriver.remote.webelement.WebElement'>
>>> search_element.click() # 
>>> search_element.send_keys('selenium' + Keys.ENTER)
>>> assert driver.current_url.endswith('/wiki/Selenium')            # 
>>>
>>> driver.find_element_by_link_text('Selenium (software)').click() # 
>>> assert driver.current_url.endswith('wiki/Selenium_(software)')  # 
>>> driver.close() #
產生 WebDriver，Firefox 的視窗會在這個時候被帶出來。
get() 開啟特定網址。
要對特定 element 進行操作前，必須先用 find_element*() 找出來並取得對應的 WebElement。
Element 直接支援 click()、send_keys() 等操作。
直接用 assert 做驗證。
close() 會關閉目前的視窗；如果要關閉所有的視窗，可以改用 quit()。

過程中，除了開啟／關閉瀏覽器之外，可以觀察到測試主要有 3 個步驟：

  1. 找出特定的 Element。

  2. 對 Element 進行操作。

  3. 驗證網頁或 Element 的狀態。
