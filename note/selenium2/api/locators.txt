:title: [Selenium][WebDriver] 定位操作的目標

[[strtegy]]
嚴格來說，目前 WebDriver 支援八種 locator strategies：

 * ID
 * Class Name
 * Tag Name
 * Name
 * Link Text
 * Partial Link Text
 * CSS
 * XPath

另外還支援執行一小段 JavaScript 傳回 DOM Element 的做法。
