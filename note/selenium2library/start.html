Selenium2Library / Getting Started

跟 SeleniumLibrary 一樣內附一個 demo，可以用來快速地體驗並幫助上手 Selenium2Library。

首先安裝 Selenium2Library，然後從官網下載 robotframework-selenium2library-master.zip，解開後切到 demo/ 目錄：

.
├── demoapp
│   ├── html
│   │   ├── demo.css
│   │   ├── error.html
│   │   ├── index.html
│   │   └── welcome.html
│   └── server.py
├── log.html
├── login_tests
│   ├── invalid_login.txt
│   ├── resource.txt
│   └── valid_login.txt
├── package.py
├── report.html
└── rundemo.py

把 SUT 執行起來：（之後可以將 start 換成 stop 停掉 SUT）

$ python rundemo.py demoapp start

手動玩看看 http://localhost:7272/html （正確的帳密是 demo/mode）。

接下來用 Robot Framework 的 start-up script (pybot) 執行 test case。你會看到 Firefox 被叫起來，自動填入正確的帳密…

$ pybot -d /tmp login_tests/valid_login.txt
