= [TortoiseGit] 安裝 =

 1. 確認系統是 Windows XP SP3+，然後到http://code.google.com/p/tortoisegit/wiki/Download[官網]下載安裝程式（`TortoiseGit-<version>-<32|64>bit.msi`）執行。
 2. 如果有需要，再下載語言包（還是有分 32/64-bit）安裝，有繁中版。
 3. TortoiseGit 雖然是架構在 link:../msysgit/msysgit.html[msysGit] 之上。從http://code.google.com/p/msysgit/downloads/list?q=full+installer+official+git[這裡]下載 Git for Windows 的安裝程式。
 4. TortoiseGit > Settings > Git > User Info 輸入 Name 跟 Email，之後 commit 會用得到。
 5. Commit 時帳密要重複輸入，要怎麼解決？
