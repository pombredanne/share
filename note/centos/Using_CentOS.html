Using CentOS

Daemons

  • chkconfig 可以用來設定或檢查在哪些 run levels 下要預設啟動。

Networking

  • system-config-network

    • 在 text mode 下也能使用

    • 修改的結果會立即反應到上述所有的組態檔 (不只是 /etc/sysconfig/networking), 但要從新啟動 network 才能生效 (service network restart 或 ifdown eth0; ifup eth0)
