EGit

EGit 是 Eclipse 的 Git Team Provider，建構在 JGit 之上（pure-Java）。

EGit 的安裝位置在：

http://download.eclipse.org/egit/updates

選擇 Eclipse Git Team Provider > Eclipse EGit 即可，其他相依的套件會自動被選進來，例如 Eclipse JGit。

Eclipse 4.2 已經內建對 Git 的支援，不需要額外安裝。

EGit 提供了 Git Repository Exploring 這個 Perspective，還有一些 Views：

  • Git Reflog

  • Git Repositories

  • Git Staging

  • Git Tree Compare
