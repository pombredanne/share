 * 1992 年出現
 * English-like language -> another language interface to the Mac
 * 除了 scripts 外，還可以寫 applets, applications, Automator actions
 * for detailed automation vs. general automation needs (Automator)
 * more than just a macro-language, make decisions based on user-interaction or by parsing and analyzing data, documents or situations.
 * 發現 AppleScript 比表面上看到的還要強大很多，甚至還可以做 GUI automation。
 * AppleScript Editor
 * hosted application?
 * support integrated into hundreds of Apple and third-party applications??
 * http://stackoverflow.com/questions/4311090/how-do-you-do-an-include-in-applescript[How do you do an include in applescript? - Stack Overflow] (2010-11-30) 
 * http://stackoverflow.com/questions/7641806/how-would-you-put-an-applescript-under-version-control[git - How would you put an applescript under version control? - Stack Overflow] (2011-10-03) - `osacompile`, `run script file theFile`
 * http://www.oatmealandcoffee.com/ApplescriptObjC/CheatSheet/[Cheat Sheet] - 提到 when linking one script file to another in code because using the .applescript extension will result in the file not being found.
 * https://github.com/dustinrue/ControlPlane/issues/168[Execute an .applescript file just like .scpt · Issue #168 · dustinrue/ControlPlane] (2012-08-22) - `.applescript` 可以用 `osascript` 執行（`#!/usr/bin/osascript`），
 * http://lists.apple.com/archives/applescript-users/2012/Jun/msg00065.html[[ANN] git-scptd 1.0.1 : Keep AppleScript bundles in the version control system "git"] (2012-06-22) - git 可以比對出 binary 的差異？
 * http://cross-the-sea.blogspot.tw/2009/11/i-recommend-text-format-for-applescript.html[Learning Programming: I Recommend Text Format for AppleScript] (2009-11-26)
 * http://www.apeth.com/sd5help/indexfolder/referencfolder/helpglossaryfolder/bytecode.html[Glossary: Bytecode] - AppleScript 會被 compile 成 bytecode，用 editor 再開起來時，其實是經歷了 decompile 的過程，所以不見得 format 會跟你當初寫的一樣。如果是 compile 成 run-only，變數名稱之類的資訊沒有被存起來，就無法 decompile 了。
