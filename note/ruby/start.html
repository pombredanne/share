Ruby / Getting Started

首先確認 Ruby 1.9.x 已經安裝妥當。

$ ruby --version
ruby 1.9.3p0 (2011-10-30 revision 33570) [x86_64-linux]
hello.rb
# say hello to such a great gem.
puts "Hello, Ruby!";

結尾的分號通常不會加，除非要把多個 statements 寫在同一行。

$ ruby hello.rb
Hello, Ruby!
hello.rb
#!/usr/bin/env ruby
# say hello to such a great gem.
puts "Hello, Ruby!";

chmod a+x hello.rb
./hello.rb
Hello, Ruby!
