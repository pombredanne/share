Linux Networking / Traceroute

traceroute 跟 tracepath 都可以用來 "查線"，看是網路斷在哪邊，或是慢在哪邊… 兩者最明顯的差別是一般使用者就可以執行 tracepath，但 traceroute 必須要有 root 權限才行。

有趣的是，Ubuntu 預設裝有 tracepath，但 traceroute 則要手動安裝 traceroute 套件才有。
