= INI =

INI 可說是 configuration files 的業界標準（de facto standard）。雖然 INI 不是一個正式的標準，但由於簡單易懂，編寫上也比 XML／JSON／YAML 等其他格式容易許多，所以應用相當普及。很多時候，XML 會顯得大材小用，INI 反而比較合適。

INI 這個名字源自 "Initialization", 常見的副檔名就是 `.ini`, 當然用 `.conf` 或 `.cfg` 的也大有人在。

TIP: INI 最早是出現在 Windows 上做為儲存系統設定之用，但 Windows 95 之後微軟力推 Windows Registry 存放組態，之後 .NET 又改採 XML `.config`。
