= Apache Ant =

== 快速上手 ==

 * 安裝
 * 走過 http://ant.apache.org/manual/tutorial-HelloWorldWithAnt.html[Tutorial: Hello World with Apache Ant]。

== 安裝 ==

 1. 到官網下載 `apache-ant-<version>-bin.tar.gz`，解開後放到 `ANT_HOME` (例如 `~/dev/ant`)。
 2. 將 `$ANT_HOME/bin` 加到 `PATH` 環境變數。

== 參考資料 ==

 * http://ant.apache.org/[Apache Ant]
 * http://ant.apache.org/manual/index.html[Apache Ant User Manual]
