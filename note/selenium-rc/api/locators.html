Selenium RC / API / Locators

  • add_location_strategy(strategyName, functionDefinition)

  • allow_native_xpath(allow)

  • assign_id(locator, identifier) - 用 CSS 或 XPath 找到 element 後，給定一個 ID，之後就能用 ID 直接參考。

  • get_css_count(css) - 注意 locator 開頭還是得加 css=。

  • use_xpath_library(libraryName)

  • highlight(locator) - 檢查 locator 的有效性。
