在多人使用的環境下，其他瀏覽器會叫用哪個使用者呢？日前發現 Mac Safari 會跟著啟動 Selenium Server 的身份走。但為什麼之前 IE 沒有這類問題？

$ java -jar selenium-server-standalone-<version>.jar -firefoxProfileTemplate <path-to-profile>
