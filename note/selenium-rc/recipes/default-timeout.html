Selenium RC / Recipes / 取得 Default Timeout

>>> selenium.get_eval('this.defaultTimeout') 
u'3000'
>>> int(float(sel.get_eval('this.defaultTimeout')) / 1000) 
3
透過 this.defaultTimeout 可以取得目前 timeout 的設定值，注意取回來的單位是毫秒（ms）。
轉成數值，這裡會多加一層 float() 的轉換是因為從 this.defaultTimeout 取回的數值可能帶有小數點，例如 6000.0。
