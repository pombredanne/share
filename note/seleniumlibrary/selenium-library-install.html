SeleniumLibrary / 安裝

SeleniumLibrary 2.9.1 跟 Firefox 有一些不相容的問題，建議安裝 2.9 即可。

通常是直接從 source 安裝：

  1. 下載 robotframework-seleniumlibrary-<version>.tar.gz（超過 20MB，因為內含 Selenium Server）

  2. 解開後執行 python setup.py install。
