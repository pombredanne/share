= Selenium Library: 跟對話視窗互動 =

跟對話視窗相關的 keyword 有：

 * `chooseCancelOnNextConfirmation()`
 * `chooseOkOnNextConfirmation()`
 * Confirm Action
 * `Choose Ok On Next Confirmation`
 * `Choose Cancel On Next Confirmation`
 * `Alert Should Be Present`
 * Get Alert Message

 * 其中 Choose Cancel On Next Confirmation 是搭配 Confirm Action 使用的。
 * 一定要用這幾個 Confirm Action, Alert Should Be Present 或 Get Alert Message 把 alert/confirmation dialg 的訊息吃下來，否則後面的 keyword 都會失敗，中間不能再安排其他 steps。
