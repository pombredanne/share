書籍

  • Addison-Wesley - PhoneGap Essentials: Building Cross-platform Mobile Apps (2012-05 est.)

  • O’Reilly - Building Mobile Applications with Java: Using GWT and PhoneGap (2012-03)

  • O’Reilly - 20 Recipes for Programming PhoneGap (2012-03)

  • O’Reilly - Building Android Apps with HTML, CSS, and JavaScript, 2nd Edition (2012-01 est.)

  • Apress - Beginning PhoneGap (2011-12 est.)

  • Wrox - Beginning PhoneGap (2011.12) - 前面花了三個章節在講 PhoneGap 的基礎，之後才依序講解 PhoneGap API 不同的面向，最後用一個多媒體記事軟體做為範例來做結尾。

  • Packt - PhoneGap Beginner’s Guide Book (2011.09) - 為什麼官網上只推薦這本書？這本書封面上就寫著 “Learn by doing: less theory, more results.”，第一章就直接進入實作，章節的安排比較適合入門。

其他文件

  • CSC Studio • 關於 PhoneGap 的三大迷思 (2012-06-18) - PhoneGap 只是一層個殼（內含瀏覽器的 App），實際的開發需要熟悉 HTML5，而且得從 App 的角度下去設計。若某些平台特性無法被發揮出來時，還是得考慮為某些平台開發 native app。
