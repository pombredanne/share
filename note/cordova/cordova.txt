= Cordova =

 * Cordova 是一個橫跨不同 mobile devices 的 JavaScript API，讓 HTML + CSS + JavaScript 開發的 web app 能夠以 cross-platform 的方式存取 device-specific/native functions。
 * 雖然 web app 不是用 native code 開發，但最終也是透過不同 platform 的 SDK 包裝成可以上架到 app store 的 package。 
 * Cordova 跟 PhoneGap 的關係是什麼？
