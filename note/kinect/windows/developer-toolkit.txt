:title: [Kinect for Windows] Developer Toolkit

Developer Toolkit 預設的安裝位置在 `%ProgramFiles%\Microsoft SDKs\Kinect\Developer Toolkit v<version>` (`KINECT_TOOLKIT_HOME`)。它是在 link:sdk.html[SDK] 之上，額外提供：

 * Code Samples - 原始碼在 `%KINECT_TOOLKIT_HOME%\Samples` 下依不同的程式語言分開存放，也很貼心地將編譯後的可執行檔集中在 `%KINECT_TOOLKIT_HOME%\Samples\bin` 下。
 * link:kinect-studio.html[Kinect Studio]
 * link:toolkit-browser.html[Toolkit Browser]
 * ...
