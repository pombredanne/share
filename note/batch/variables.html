Batch: 變數（Variables）

變數在使用前要先用 set 定義過：

C:\> set name=value

使用時則透過 %name% 的表示法來引用：

C:\> echo %name%
value

如果要判斷某個變數有沒有被定義過，可以用 if defined 來檢查：

C:\> if defined name (echo name = %name%) else (echo name is not defined)
name = value

C:\> set name=
C:\> name is not defined

如過引用到的變數沒有定義過，並不會發生錯誤，只是 `%name% 不會展開而已：

 C:\> set name=
 C:\> echo %name%
 %name%
