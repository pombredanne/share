Batch: CALL

call 可以從 batch file 呼叫執行另外一支 batch file，呼叫端跟被呼叫端都執行在相同的環境下（不會引發另一個 process）。例如 script_a.bat 跟 script_b 都需要用到 env.bat 裡設定的環境變數：

script_a.bat
echo script_a
set VAL=val_a
call env.bat
echo %VAL%
env.bat
set VAL=val_env

測試結果：

C:\tmp>script_a.bat

C:\tmp>echo script_a
script_a

C:\tmp>set VAL=val_a

C:\tmp>call env.bat

C:\tmp>set VAL=val_env

C:\tmp>echo val_env
val_env
