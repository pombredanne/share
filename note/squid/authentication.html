Squid Authentication

  • ncsa_auth 認證可以直接吃 htpasswd 所製作的密碼檔。

auth_param basic program /usr/lib64/squid/ncsa_auth /etc/squid/squid_user.txt auth_param basic children 5 auth_param basic realm Welcome to VBird’s proxy-only web server

acl squid_user proxy_auth REQUIRED

http_access allow squid_user
