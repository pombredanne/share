Ubuntu / Mouseless

  • Super - 也就是 Windows 國旗鍵。輕按一下會叫出 Dash，長按則會出現 Keyboard Shortcuts 的提示畫面。

  • Ctrl + Alt + L - 鎖定螢幕。

  • Ctrl + Super + D - 顯示桌面；再按一次可以回復。

  • Ctrl + Super + Up - 將目前的視窗最大化。

  • Ctrl + Super + Down - 回復一般視窗大小或最小化。不過目前最小化好像有問題?? 可以用 Ctrl + Alt + Num-0 達成。

  • Ctrl + Super + Left/Right - 將目前的視窗放大後擺在畫面左半邊或右半邊。

  • Super + S - 綜覽所有的桌面；接著可以用方向鍵 + Enter 切換桌面。

  • Ctrl + Alt + (Shift) + Left/Right/Up/Down - 切換桌面。加 Shift 時可以同時移動目前的視窗。

  • Super + W - 綜覽目前桌面所有的視窗。但好像不能用方向鍵選取??

  • Alt + Tab - 在目前桌面的兩個應用程式間切換；長按可以選取其他應用程式（同樣侷限在目前的桌面)。

  • Alt + ` - 在同一個應用程式的兩個視窗間切換；長按可以選取其他視窗（同樣侷限在同一個應用程式）。

  • Super + (Shift) + 0..9 - 開啟或切換到 Launcher 上的應用程式，不論它的視窗是否在目前的桌面。

    長按 Super 時，Launcher 上的 icon 會被標示數字，由上到下是 1 → 9 → 0。利用這個特性，只要事先將 icon 的位置做適當的安排，就可以快速在幾個常用的應用程式間快速切換。

    過程中如果在加上 Shift 鍵，會開啟另一個新的 instance。其實用滑鼠操作用，搭配 Shift 也可以有相同的效果。

  • Super + A - 搜尋應用程式。

  • Super + F - 搜尋檔案目錄。

  • Alt + F1 - 調出 Launcher，然後用上下鍵做選取（右鍵可以調出 quicklist），最後按 Enter 確認。

  • Ctrl + Alt + T - 開啟一個新的 Terminal 應用程式。

  • Super + T - 垃圾桶。

  • Alt - 輕按一下，畫面上方會出現 HUD (Head-Up Display)；長按則會叫出應用程式的選單。

  • Alt + Space - 開啟視窗本身的選單（不同於應用程式的選單）。

  • (Shift) + PrtScn - 拍攝整個桌面的截圖。加 Shift 時只拍目前的視窗。

另外檔案開啟／儲存的對話視窗有些專用的快捷鍵：

  • Ctrl + L - 顯示或將游標移到 Location 欄位，可以直接輸入檔案路徑。

  • Left / Right - 將焦點在對話框左右兩側切換，之後可以用上下鍵切換。

  • Enter / Backspace - 進入下一層或回上一層目錄。
