Anki / Installation

Anki 內附在 Ubuntu 的版本通常比較舊（例如 Ubuntu 10.10 搭載 Anki 0.9.9.8，但最新版已經來到 1.2.8）。建議從官網下載 anki_<version>_all.deb 安裝：

$ sudo dpkg -i anki_<version>_all.deb

(Reading database ... 317103 files and directories currently installed.)

  Package python-sqlalchemy is not installed.

  Package python-beautifulsoup is not installed.

 anki

$ sudo apt-get -f install

安裝 .deb。
如果安裝 .deb 的過程中有許多 dependency 沒有被滿足，可以用 apt-get 來補齊。

之後可以從 Applications > Education > Anki 啟動 Anki。
