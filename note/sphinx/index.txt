= Sphinx =

應用實例：

 * 很多網站是用 Sphinx 寫的
 * Python http://docs.python.org/index.html[官網]的文件，一整個就是用 Sphinx 寫的。
 * 在http://docs.python.org/tutorial/datastructures.html[這裡]看到程式碼區塊右上角有個 ">>>" 按鈕，按下去會將 output 跟 ">>>" 都拿掉，還滿方便的，看來 Python 文件用 Sphinx 產生會比 http://epydoc.sourceforge.net/[Epydoc] 來得好？
 * http://readthedocs.org/[Read the Docs] 的文件也是用 Sphinx 來寫。
 * http://www.worldhello.net/gotgithub/[GotGitHub — GotGitHub] 這本書也是用 Sphinx 寫的。
