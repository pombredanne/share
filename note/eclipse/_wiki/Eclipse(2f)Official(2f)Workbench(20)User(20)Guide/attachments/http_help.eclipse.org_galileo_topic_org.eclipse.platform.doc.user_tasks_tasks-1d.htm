Workbench User Guide > Tasks Problems, bookmarks, tasks and other markers Markers are objects that may be associated with Workbench resources. There are many uses of markers in the Workbench, including providing support for bookmarking resources or locations within resources, tracking ongoing tasks, or displaying error messages. See the related tasks section for more details.
