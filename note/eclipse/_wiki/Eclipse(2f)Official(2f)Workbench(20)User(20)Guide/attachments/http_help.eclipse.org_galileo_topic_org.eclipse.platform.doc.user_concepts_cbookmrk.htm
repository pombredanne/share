Workbench User Guide > Concepts Bookmarks You can place an "anchor" either on a resource within the Workbench, or at a specific line within a file, by creating a bookmark. Then you can use the Bookmarks view to return to those files quickly. The Bookmarks view (Window > Show View > Bookmarks) displays all bookmarks that you have created.
