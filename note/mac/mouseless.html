Mac / Keyboard Shortcuts

  • Ctrl + Command + F - 全螢幕。

  • Command + A - 全選。

  • Command + C - 複製。

  • Command-V - 貼上。

  • Command-Tab - 在兩個應用程式間切換。

  • Command-F5 - 開關 VoiceOver 的功能。

  • Ctrl-Left/Right - 切換桌面。

  • Ctrl-A, Command-Left - 將游標移到行首。

  • Ctrl-E, Command-Right - 將游標移到行尾。

  • Command-UP, Fn-Left - 將游標移到檔案的開頭，相當於 Home。

  • Command-Down, Fn-Right - 將游標移到檔案的結尾，相當於 End。

  • Fn-Up / Down - 上下翻頁。

  • F11 - 顯示桌面。

  • Shift-Command-G - Go to Folder；在應用程式裡開啟檔案時，也可以用這個組合鍵直接輸入完整的路徑名稱，不用一層一層選。

  • Command-W - 關閉目前的視窗

  • Command-Q - 離開目前的應用程式；不只是按下叉叉，而是 Quit。
