Mac OS X: 設定 PATH 環境變數

有別於 其他環境變數 的設定方式，Mac 全域 (system-wide) 的 PATH 其實是設定在 /etc/paths，內容如下：

/usr/bin
/bin
/usr/sbin
/sbin
/usr/local/bin

如果要增加其他路徑的話，建議透過 /etc/paths.d/ 底下的檔案做新增。例如：

$ cat /etc/paths.d/50-X11 
/usr/X11/bin              
$ echo $PATH
/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/bin:/usr/X11/bin
檔名前面的數字應該是用來控制串接的順序。
檔案裡的每一行代表一個路徑。
