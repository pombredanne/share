Mac OS X

認識 Mac OS X

  • 1999 推出最後一版 class Mac OS - Mac OS 9，之後就開始推 Unix-based OS - Mac OS X。

  • 雖然 Mac OS X 是繼 Mac OS 9 的下一個版本，但其中的 X 是指基於 NeXT 所發展出來的 Unix-based graphical operating system。因此 X 有人唸成 /ˈtɛn/，但強調它是 Unix-based 時也有人唸做 /ˈɛks/（試想，Mac OS X 版號如果跳到 11 時，唸做 /ˈɛks/ 應該比較合理吧？）

  • Mac OS X 的第一個公開的版號是 10.0，之後以 10.1, 10.2 的方式增加。每一個版號都有對應的 codename（以大型猫科動物 bit cat 來命名），之後市場也習慣用 codename 來行銷… 幾個比較知名的版本是 10.4 Tiger, 10.5 Leopard, 10.6 Snow Leopard, 10.7 Lion。

  • 一開始 Mac OS X 還另外推出了 server edition（架構與 desktop 版本一樣，只是多了方便管理多台機器的工具而已），但 10.7 Lion 開始就沒有額外推出 server edition，改以 add-on package 的方式在 Mac App Store 上販售。

  • Mac OS X 10.5 Leopard 是最後一個支持 PowerPC 架構的版本，10.6 Snow Leopard 就只支持 Intel 架構的機器，10.7 Lion 更要求至少 Intel Core 2 Duo。

    • 10.5.7 是 10.5 Leopard 的最後一個版本，

  • OS X Lion 在 Mac App Store 只賣 US$ 29.99，大約 1000 塊台幣，算滿便宜的。
