Mac / Windows 鍵盤

如果鍵盤並非專為 Mac OS 設計，可能沒有某些標準按鍵：

  • Command 可以用 Windows 國旗鍵代替。

  • Option (⌥) 可以用 Alt 代替。

  • Control 就用原來的 Ctrl 鍵即可。

Windows 下常用的組合鍵 Ctrl+A, Ctrl+C … 等，在 Mac 都改成 Command+A 或 Command+C 等，所以不是用 Mac 鍵盤的話，要使用國旗鍵才對。另外滑鼠捲動的方向跟 Windows/Linux 不同，需要花一點時間適應。
