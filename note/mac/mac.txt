 * 執行在 iPhone, iPod, iPad 上的 iOS，也是從 Mac OS X 衍生出來的。
 * Apple stated that Mac OS X would not run on Intel-based personal computers aside from its own, but a hacked version of the OS compatible with conventional x86 hardware was developed by the OSx86 community.
 * Mac OS X's core is a POSIX compliant operating system (OS) built on top of the XNU kernel, with standard Unix facilities available from the command line interface. Apple has released this family of software as a free and open source operating system named Darwin. On top of Darwin, Apple layered a number of components, including the Aqua interface and the Finder, to complete the GUI-based operating system which is Mac OS X.
 * 第一次開機時 Apple ID 可以不用給，`@mac.com` 或 `@me.com` 的 email 帳號可以直接當做 Apple ID。
 * 建立帳號時要提供 full name 跟 account name，其中 account name 會成為 home folder，所以事後不能變更。
 * 從 Safari 上下載檔案時，不知道為什麼 `.zip` 下載回來會自動解開，而 `.tar.gz` 下載回來則會變成 `.tar`。
 * 需要重新安裝整個 Mac OS X，只要在開機時按住 Command + R，最後就會帶出 Mac OS X Utilities 的畫面
   ** http://support.apple.com/kb/HT4718[OS X Lion: About Lion Recovery]
   ** 進 Disk Utility 將 Macintosh HD 給清掉 (Erase)
   ** 回到 Mac OS X Utilities，選 Reinstall Mac OS X 即可，還原到 Macintosh HD。
   ** http://docs.info.apple.com/article.html?path=Mac/10.7/en/mh27903.html[Mac OS X 10.7 Help: Erase and reinstall Mac OS X]
   ** 如果 Erase Macintosh HD 後沒有 Reinstall Mac OS X 成功，開機時會遇到 "資料夾裡大大的問號" 的畫面，只要接上網路開機，就可以開始 Start Internet Recovery。這個動作應該只是下載 recovery 而已，大概 20 分鐘，所以完成後還是要繼續做 Reinstall Mac OS X。不過之後發現想直接做 Reinstall Mac OS X 時，卻得到 "Could not find installation information for this machine. Contact AppleCare."
 * Apple > Software Updates... 有找到更新時，可以從 Show Details 看到細節。這裡的更新包括 iTunes、Mac mini EFI Firmware Update、Safari、AirPort Utility 等。
 * Mac 習慣用資料夾的 '副檔名' 來識別特殊的用途，例如 `.app` 或 `.vmwarevm`
 * Mac mini 內建揚聲器，第一步選擇語言時就會播放語音說明...4
 * 安裝過程中有 VoiceOver 的輔助
   ** The VoiceOver Quick Start 會簡短說明幾個重要鍵的位置
   ** VoiceOver commands，也就是 VO keys，固定以 Control+Option 開頭再加上其他鍵的組合，例如 "VO-Right Arrow" 指的就是 Control+Option+Right Arrow 的組合。
 * VO-K 可以叫出 Keyboard Help，按 Escape 可離開。在這裡可以測試非 Mac OS 專用鍵盤的每個鍵，是對應到標準鍵盤的哪個鍵。
