iOS / Tools / iPhone Configuration Utility (iPCU)

iPCU 從 App Store 上找不到，必須到這裡下載 iPhoneConfigUtility.dmg 手動安裝。

要安裝 .ipa 的話，將 .ipa 先加到 LIBRARY > Applications，然後再到 DEVICES > Your Device > Application 按 Install 即可。不過它不支援 Update，必須要先做 Uninstall 再 Install 才行，這不太符合測試上的需求，但 iTunes 就沒這個問題了。
