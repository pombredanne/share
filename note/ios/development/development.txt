:title: [iOS] Developement

 * Mac OS X 10.7 Lion
 * Apple Developer -> download SDK
 * Objective-C + Cocoa Touch
 * Xcode，內含 SDKs for OS X and iOS。
 * iOS SDK
   ** simulator, no hardware-dependent featrues like accelerometer and camear.
   ** simulator 只有最新版，其他舊版要另外下載，有 4.3, 5.0 跟 5.1
 * project templates
 * storyboards
 * iCloud
 * table views
 * Automatic Reference Counting (ARC)
 * developer options
   ** free - simulator, 不能將 application 下到 device 上執行，也不能發佈到 App Store。
   ** iOS Developer Program (Individual/Company) - $99/year
   ** iOS Developer Enterprise Program - $299/year，除了多 developers 外，也可以發佈 in-house apps。

事實上，要開發 iOS app 不一定得用 Objective-C 寫才行：

 * http://www.rubymotion.com/[RubyMotion - Ruby for iOS]
 * http://xamarin.com/monotouch[MonoTouch - Create amazing iOS apps with C# and .NET]
 * link:../../phonegap/phonegap.html[PhoneGap]
 * http://www.sencha.com/products/touch/[Sencha Touch]
 * ...
