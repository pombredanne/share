P4 / Clients / P4V Visual Client

安裝

  1. 下載 P4V，支援 Windows、Mac、Linux 等。以 Windows 的 p4inst.exe 為例。

  2. 預設會安裝 PV4、P4Merge、P4Admin 跟 P4。

  3. 設定 server 的位置跟登入帳號。

  4. 開啟 Start > Perforce > P4V。
