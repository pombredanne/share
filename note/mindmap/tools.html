繪製心智圖的工具

知名的工具有：

  • XMind - 畫面質感不錯 (還可以畫魚骨圖、組織圖等), 但效能還有很大的進步空間。

  • FreeMind - 雖然畫面比不上 XMind，但用起來還是比較順手。

    • SciPlore MindMapping

    • Freeplane

  • Text 2 Mind Map - Simple mind mapping online - 可以將 outline text 轉成 mind map（可下載成 PDF 或圖檔，甚至直接分享到 FB 或 Twitter）

  • MindVisualizer - 也可以將 plain text 轉成 mind map。

  • Mappio - Mind Map Library - 上傳 text 可以轉成 mind map。

  • Compendium - 比較偏 Concept Mapping 的應用

  • Semantik - 用 Mind Map 幫助學生或其他專業人士產出複雜的文件; 可以從 Mind Map 轉出 S5 Presentation、OpenDocument Text/Presentation 等. (Ubuntu 下只要安裝 semantik 套件即可)

  • iMindMap - 心智圖發明人 Tony Buzan 領導開發的產品

  • MindManager - 市場心智軟體的領導廠商

  • MindMeister - 最低月付 4.99 美金。

  • Mindomo - 最低月付 6 美金。

  • bubbl.us

  • Mind42.com - 免費

  • Mapul

  • WiseMapping

  • mindmaps
