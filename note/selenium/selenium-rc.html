Selenium / Remote Control (RC)

Selenium Remote Control (RC) 架構在 Selenium Core 之上，由兩個部份組成：

  • Client Driver - 支援不同的程式語言，透過 Selenium (RC) API 來跟 Server 溝通。

  • Selenium (RC) Server - 直接操控瀏覽器，也做為 Client Driver 跟瀏覽器溝通的橋樑。
