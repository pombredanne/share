Selenium（硒） - Web Browser Automation

Selenium 主要用來自動化測試 Web 應用程式，但也可以將某些需要透過網站來完成的工作給自動化。

Selenium 2 指的就是 WebDriver，而 Selenium 1 指的則是 Selenium RC

Selenium 有時簡稱做 SEL，例如 Selenium 2.0 等同 SEL 2.0
