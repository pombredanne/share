Selenium / RC / 操作表單控制項

有些 API 通用於不同的表單控制項：

  • is_editable(locator) - 從字面上看來是檢查可否編輯，但其實是檢查 form element 沒有被 disable。

  • get_value(locator) - 取得輸入控制項的 value，也就是 value 屬性的值。

  • type(locator, value) - 直接設定控制項的文字內容（也就是 value 屬性的值），不會引發 keystroke event，需要的話得改用 type_keys()。

其他還有：

  • Checkbox

  • get_all_buttons()

  • get_all_fields()

  • get_cursor_position(locator)

  • set_cursor_position(locator, position)

  • submit(formLocator)
