Selenium / RC / 取得文字內容

跟取得文字內容相關的 API 只有一個：

get_text(locator)

以下面的 HTML 為例：

<html>
<body>
<div id="msgbox">
<h4><img src="icon.png"/>title</h4>
<span>description</span>
</div>
</body>
</html>
>>> sel.get_text('id=msgbox')
u'title description' 
>>> sel.get_text('css=#msgbox h4')
u'title'
>>> sel.get_text('css=#msgbox span')
u'description'
get_text() 會取出底下所有使用者看得到的文字內容，注意 title 跟 description 中間有個空白。
