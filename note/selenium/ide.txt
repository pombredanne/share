:title: Selenium-IDE

到官網 http://seleniumhq.org/download/[Downloads 頁面]的 Selenium IDE 點選 `selenium-ide-<version>.xpi` 的連結下載安裝，會同時安裝下面五個項目：

 * Selenium IDE
 * Selenium IDE: Ruby Formatters
 * Selenium IDE: Python Formatters
 * Selenium IDE: C# Formatters
 * Selenium IDE: Java Formatters
