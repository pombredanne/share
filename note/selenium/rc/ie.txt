= Selenium RC: Internet Explorer =

透過 RC API 要開啟 IE 時，'browser' 要傳入 `*iexplore`。

常見的問題有：

 * link:ie-secure_nonsecure_items.html[This page contains both secure and nonsecure items]。
 * link:ie-download-blocking.html[透過 Selenium 下載檔案時，IE 會出現黃色警語阻擋下載]。
 * link:ie-permission-denied.html[ERROR: Command execution failure: Permission denied]
 * link:ie-invalid-argument.html[ERROR: Command execution failure: Invalid argument]
 * link:ie-script-null-error.html[Selenium 1/RC + IE 開新窗時出現 Script Error - 'null' is null or not an object]

Pop-up Blocker 一開始是出現在 IE 7，預設是開啟的。阻擋到 pop-up 時，畫面上方會出現一條 information bar，點下去跳出選單：（奇怪的是，手動按並不會跳出這個 blocker？）

.Pop-up blocked. To see this pop-up or additional options click here...
image:Using_Selenium.files/images/ie_popup_blocked.png

如果不影響測試，只要從 Tools > Pop-up Block 選 Turn Off Pop-up Blocker 即可。
