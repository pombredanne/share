:title: [Team Viewer] Installation

== Ubuntu ==

 1. 從官網下載 `teamviewer_linux[_x64].deb`（約 18 MB）。
 2. 執行 `sudo dpkg -i teamviewer_linux.deb`。
 3. 開啟 Applications > Internet > TeamViewer 7

== Mac OS X ==

 * 從官網下載 `TeamViewer.dmg`，執行後會解開 `TeamViewer.pkg`，再執行即可開始安裝。
