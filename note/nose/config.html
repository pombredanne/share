nose: 組態（Configuration）

nosetests 除了可以從 command line 指定參數之外，也可以事先將某些參數寫在 ~/.noserc 或 ~/.nose.cfg 裡。例如：

$ nosetest --verbosity=3

可以改寫成：

~/.noserc 或 ~/.nose.cfg
[nosetests] 
verbosity=3
~/.noserc 或 ~/.nose.cfg 的格式都是 INI。
一行一個設定，跟 command line 不同的地方只是把前面的 -- 拿掉而已。
