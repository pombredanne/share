:title: [RTM] 用鍵盤操作 RTM

TIP: 在 Chrome 下搭配 link:../chrome/mouseless.html#deadmouse[DeadMouse]，可以快速在不同 tag、list 間切換。

 * Ctrl + Shift + / - 將 cursor 移到 search box （按 Tab 跳離，否則其他快捷鍵無法作用）

當 cursor 不在 search box 或其他 input fields 時：

 * Ctrl + Shift + Left / Right - 切換到上下一個分頁。
 * t - 新增一項任務 (add a new **t**ask)
 * z - 取消上一個動作
 * a - 全選 (select **a**ll)
 * n - 取消選取 (select **n**one)
 * j - 將 cursor 移到下一個 item
 * k - 將 cursor 移到下一個 item
 
下面這些快捷鍵，只作用在 cursor 所在的 item：

 * i - 選取或取消選取目前 cursor 所指的 item

下面這些快捷鍵，只作用在被選取的 items：

 * r - 變更任務名稱 (**r**ename)
 * d - 修改到期日 (**d**ue **d**ate)
 * c - 標示為已完成 (**c**omplete)
 * s - 變更 tags
 * p - 延期 (**p**ospone)
 * f - 修改 repeat 的週期 （**f**requency）
 * y - 增加一個 note
 * 1 - Priority **1**
 * 2 - Priority **2**
 * 3 - Priority **3**
 * 4 - No priority
