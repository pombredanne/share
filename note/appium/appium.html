Appium

Appium 唸做 `æpiәm，要瞭解它是什麼，建議先看過 GTAC 2013: Appium: Automation for Mobile Apps 的介紹。

簡單地說，它讓你可以透過同一套 API (Selenium 2/WebDriver API)，操控 Android、iOS 跟 Firefox OS 設備（或模擬器），藉以實現手機的自動化測試。

目前看似在勢頭上，包括 Source Labs、 TestingBot 等，都已採用 Appium 做為 cloud-based mobile testing 的解決方案。

首先安裝 Appium、啟動 Appium Server，接著就可以開始 Android、iOS 或 Firefox OS 的測試…
