:title: Java SE (Standard Edition)

Java SE (Java Platform, Standard Edition) 的安裝分成 JRE 跟 JDK 兩部份；執行 Java 程式時要用到 JRE (Java Runtime Environment)，開發應用程式的話則會用到 JDK (Java Development Kit)。
