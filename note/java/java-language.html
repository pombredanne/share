Interface

  • Interface 的名稱不用刻意冠上一個 I。寫程式時儘可能用 interface 來參照，所以前面冠個 I 反而降低了程式的可讀性。

  • Interface naming in Java - Stack Overflow (2009-02-12) - 很大一部份的人都同意 interface 的名稱不用刻意冠上一個 I。
