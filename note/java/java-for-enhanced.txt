= [Java][Language] Enhanced For Loop =

JDK 1.5 開始支援 `for` 的加強版（做為一個 syntax sugar），類似其他語言 for each 的用法，可以更容易地遶行 collection 跟 array：

 for (type item : items) {
     statement(s)
 }
