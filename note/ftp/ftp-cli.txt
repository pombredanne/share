:title: [FTP] ftp CLI

`ftp` -- Internet file transfer program

如果要透過 `ftp://` 直接下載檔案，各平台的用法差異滿大的：

 * FreeBSD/Mac: `ftp://[user[:password]@]host[:port]/path[/]` （給定 'user' 但沒有 'password' 時會提示輸入）
 * Debian - 不支援。
