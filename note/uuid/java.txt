:title: [UUID] 用 Java 處理 UUID

在 Java 裡可以用 `java.util.UUID` 來處理 UUID。

如果手邊有個 UUID，可以用 `fromString()` 轉成 `UUID` 物件：

 UUID uuid = UUID.fromString("00001101-0000-1000-8000-00805F9B34FB");
