iRise / Studio MX - Go beyond prototypes and wireframes

  • 支援 Windows XP/Vista/7 跟 Mac OS X 10.7+。

  • 強調 full experience，包括 navigation、media、rich interactions、logic、sample data…

  • 產生 iDoc 文件，可以用免費的 iRise Reader 檢視並收集回饋，也可以透過免費的 iRise Mobile 在 iPhone/iPad 上實際體驗。

  • 透過 iBloc API 跟 HTML/CSS/JavaScript 可以自行擴充 widgets 跟 behaviors。

  • 中文字變成方塊字是因為字型的關係，換掉字型就好。不知道換過字型之後，在 Device 上表現會不會異常？

  • iRise Studio MX 跟 iRise Mobile 都是免費的，不知道它的營利模式是？還是只靠 enterprise solution 賺錢？

  • 免費版本的 Studio MX 同時間只能有 3 個 active project（可以先 export 成 .idoc），而且一個 project 下最多只能有 10 的頁面。

    • 提供帳密（不用申請？）就會同時將 3 個 active projects 同時分享出去？

    • 如果有超過 10 個頁面的需求，或許可以考慮拆成多個 projects

參考資料

  • iRise Studio MX 閱讀

    • 不到 3 分鐘的簡短介紹，可以加入簡單的邏輯（不需要寫程式）。

    • Effective teams eliminate mistakes before development begins.

    • 很衷肯地提到書面文件有它的適用性，但前提是客戶要能夠對最後的開發結果有完整的體認。
